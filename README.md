

# Cybertraining for junior marine science researchers at Wilmington, North Carolina

This is the repository for the web pages of Cybertraining Workshop for junior marine science researchers hosted at UNC Wilmington. This program is funded by the National Science Foundation grants OAC 2230046 Please visit https://uncwcybertraining.github.io/ for more information.

We forked this repo from https://github.com/gt-rl/gt-rl.github.io so all credit goes to the organizers of the Geometric and Topological Representation Learning (GTRL) ICLR workshop.
